# Mahmoud Hossam | Cybersecurity Portfolio

Modern, responsive personal portfolio website for **Mahmoud Hossam Abd ELMoaty** (Cybersecurity Engineer), built with **vanilla HTML, CSS, and JavaScript**.

## Overview

This portfolio showcases:

- Professional summary and education
- Skills & expertise
- Offered cybersecurity services
- Projects & portfolio work
- Certificates & courses
- Contact section with direct email/social links

The UI includes dark mode, smooth scrolling, reveal animations, interactive project modals, and mobile-first responsive behavior.

## Tech Stack

- **HTML5**
- **CSS3** (custom properties, responsive layout, animations)
- **JavaScript (Vanilla)**
- **Font Awesome** for icons
- **Google Fonts** (Inter, JetBrains Mono)

## Getting Started

No build tools or dependencies are required.

1. Clone/download this repository.
2. Open `index.html` in your browser.

For development, use VS Code Live Server for auto-reload.

## Customization

- **Personal data & dynamic content**: edit `siteData` in `js/main.js`
	- name, title, email, phone, social links
	- skills, projects, certificates
- **Layout/content sections**: edit `index.html`
- **Theme/colors/spacing**: edit CSS variables in `css/styles.css`
